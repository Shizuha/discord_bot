# discord_bot

▷ 프로젝트 설명 

• Discord 라는 VoIP 프로그램에서 제공하는 봇이라는 채팅으로 명령을 받아 그 명령을 수행하는 프로그램입니다. (Discord 에서 제공하는 Java Discord API로 만듬)

▷ 프로젝트 인원

• 주사위 기능

• 랜덤한 단어 선택 기능


▷ 사용 기술

• Front - Discord

• Back - JAVA, Gradle Project
